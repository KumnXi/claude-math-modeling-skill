# 📐 Claude Code 通用数学建模求解工作流

> 专为Claude Code打造的全流程数模求解技能，支持华中杯、国赛、美赛等比赛，一键启动从问题拆解到报告撰写的完整工作流。

---

## ✨ 技能亮点
- **全流程自动化**：自动完成问题理解、数据处理、代码实现、结果分析、可视化与报告撰写
- **多题型适配**：支持VRP路径优化、运筹规划、预测建模等数模高频题型
- **双模式触发**：支持命令快捷调用，也支持文字描述触发，避免识别失败
- **标准化输出**：自动生成规范的项目目录结构、可运行代码和论文级分析报告

---

## 📥 安装方法（两种方式，任选其一）

### 方式1：GitHub手动安装（推荐）
1.  下载仓库中的 [`SKILL.md`](https://github.com/KumnXi/claude-math-modeling-skill/blob/main/SKILL.md) 文件
2.  把文件放到对应目录：
    - **项目级生效（仅当前项目可用）**：复制到项目目录下的 `.claude/skills/` 文件夹（没有则新建）
    - **用户级生效（所有项目可用）**：复制到 `C:\Users\你的用户名\.claude\skills\` 目录下
3.  重启Claude Code，技能自动加载生效

### 方式2：skill.sh一键安装（后续发布后可用）
待上传到[skill.sh](https://skill.sh/)社区后，可直接在页面点击「Install」一键安装，无需手动复制文件。

## 🚀 使用方法

### 1. 准备工作
把你的比赛题目文件（PDF/Word）和相关数据文件（Excel/CSV）放到当前项目目录下，方便技能读取。

### 2. 触发技能（两种方式）
#### 方式A：命令快捷触发（稳定生效）
在Claude Code输入框中输入：
/math-modeling-solver

#### 方式B：文字描述触发（兼容性更强）
直接输入以下内容，无需依赖命令识别：
帮我启动通用数学建模求解工作流，处理我的数模比赛题目

### 3. 按提示补充信息
技能启动后，会自动引导你补充比赛题目、数据文件和特殊要求，按提示操作即可开始求解。

---

## 📋 工作流说明
技能会严格按照以下流程推进，确保求解过程规范完整：
1.  **阶段1：理解问题**
    读取题目与数据，拆解子问题目标、约束条件，提取所有模型参数（如车速、惩罚率、成本系数等）
2.  **阶段2：方案设计**
    自动生成标准化项目目录，确定建模方法与求解流水线
3.  **阶段3：代码实现**
    生成可运行的Python求解代码（仅使用numpy/pandas/scipy等常用库），支持VRP优化、路径规划等场景
4.  **阶段4：结果分析**
    自动生成成本构成、路径对比、灵敏度分析等论文级可视化图表
5.  **阶段5：报告撰写**
    输出符合比赛格式的求解结果分析报告，可直接用于论文撰写

## 📂 自动生成的项目结构
技能会自动帮你整理以下目录，无需手动创建，让项目文件井井有条：

```text
your-project/
├── problem_A/          # 子问题A的求解文件
│   ├── code/           # Python求解脚本
│   ├── data/           # 题目数据文件（Excel/CSV）
│   ├── figures/        # 结果可视化图表
│   └── report/         # 求解分析报告
├── problem_B/          # 子问题B的求解文件（结构同上）
└── problem_C/          # 子问题C的求解文件（结构同上）
```

## ❓ 常见问题
1.  **命令触发提示「Unknown command」怎么办？**
    改用文字描述触发方式，或检查文件是否正确放到`.claude/skills/`目录，重启Claude Code即可。
2.  **读取Excel文件时出现编码错误？**
    技能会自动处理编码问题，若仍报错，可手动将Excel保存为UTF-8编码格式。
3.  **求解过程中上下文超限怎么办？**
    输入 `/compact` 命令压缩对话，或分阶段执行工作流（先处理问题1，再处理问题2）。
